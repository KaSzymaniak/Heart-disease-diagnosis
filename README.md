# Diagnozowanie Chorob Serca

Projekt przedstawia analize danych medycznych oraz budowe modelu uczenia maszynowego do wspomagania diagnozowania chorob serca.

## Co robi projekt

- wczytuje i przygotowuje dane do analizy,
- wykonuje podstawowa eksploracje danych (EDA),
- trenuje model klasyfikacyjny do przewidywania ryzyka choroby serca,
- ocenia jakosc modelu przy uzyciu standardowych metryk,
- prezentuje wyniki i wnioski w notatniku Jupyter.

## Pliki

- `Diagnozowanie Chorob Serca.ipynb` - glowny notatnik z kodem, analiza i wynikami.

## Uruchomienie

1. Otworz notatnik w Jupyter Notebook lub VS Code.
2. Uruchamiaj komorki po kolei od gory.
3. W razie potrzeby doinstaluj brakujace biblioteki Python.

## Cel

Celem projektu jest pokazanie pelnego procesu: od przygotowania danych, przez trenowanie modelu, az po interpretacje wynikow wspierajacych diagnoze chorob serca.
